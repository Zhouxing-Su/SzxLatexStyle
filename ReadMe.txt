To make TexLive recognize user macros (styles or classes):

Add a line of "TEXMFHOME = X:/Path/To/This/Directory" in "C:/texlive/2016/texmf.cnf", 
or type command "tlmgr conf texmf TEXMFHOME X:/Path/To/This/Directory".
Organize the TEXMFHOME directory structure as TDS.

Reference: 
http://tug.org/texmf-dist/doc/texlive/texlive-en/texlive-en.html#x1-360003.4.6
